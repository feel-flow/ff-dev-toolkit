# 変更影響度評価の依頼（サンプル入力）

以下の変更について `/assess-impact` の評価を行ってください。

## 変更概要

既存の REST API `POST /api/tasks` に、**オプショナル**なリクエストパラメータ
`priority`（省略可能・既定値あり）を追加する。

## 補足（変更の波及に関する事実）

- 既存クライアントは `priority` を送らなくても従来どおり動作する（後方互換あり）。
- 既存のエンドポイント設計・データモデルの枠組みは維持したまま、要素を 1 つ足す。
- ハンドラ関数とバリデーションに新規フィールドを追加するため、
  API 仕様書とコードには波及する。

## 期待するアウトプット

`/assess-impact` の「変更影響度評価」出力形式で回答してください。
