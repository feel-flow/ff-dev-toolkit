# 変更影響度評価の依頼（サンプル入力）

以下の変更について `/assess-impact` の評価を行ってください。

## 変更概要

認証方式を、現行の **セッション Cookie ベース**から **JWT ベース**へ全面的に
切り替える。既存のセッションストアは廃止する。

## 補足（変更の波及に関する事実）

- 既存の認証ミドルウェア・トークン検証・ログイン API を作り直す必要がある。
- 発行済みセッションは JWT へ移行できず、全ユーザーの再ログインが必要（後方互換なし）。
- 認証に依存する全エンドポイントへ波及し、ロールバックには旧セッション基盤の復旧を要する。

## 期待するアウトプット

`/assess-impact` の「変更影響度評価」出力形式で回答してください。
HIGH の場合は必須手順の判定結果まで含めてください。
