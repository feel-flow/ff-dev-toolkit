# 変更影響度評価の依頼（サンプル入力）

以下の変更について `/assess-impact` の評価を行ってください。

## 変更概要

`docs/MASTER.md` の「ビルド・テストコマンド」節にあるステータス表記を、
`ready` から `stable` へ 1 箇所変更する（表現の統一のため）。

## 補足（変更の波及に関する事実）

- この `ready` という文字列は、CI スクリプト `scripts/gate.sh` が
  `grep -F "ready"` で解釈しており、リリースゲートの合否判定に使われている。
- 文字列を `stable` に変えると、`gate.sh` 側の突き合わせ対象も変更しない限り
  ゲートが常に不合格になる。
- 変更するのは MASTER.md の 1 行のみ（見た目は軽微な文言修正）。

## 期待するアウトプット

`/assess-impact` の「変更影響度評価」出力形式（分類・影響度・判定理由・
影響を受ける文書・推奨アクション）で回答してください。
