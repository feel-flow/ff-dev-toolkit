# PROJECT.md — Compass

## ビジョン

意思決定の背景を失わず、後から参加したメンバーも判断の流れを理解できるようにする。

## 対象ユーザー

- 5〜30 名のプロダクト開発チーム
- 議論と決定の履歴を残したいリーダー

## 主要機能

- 決定ログの作成
- 根拠リンクの添付
- 更新履歴の表示

## 非機能要件

- 決定ログ一覧の表示は p95 200ms 以内
- 監査ログは 1 年保持

## スコープ外

- チャット機能
- 外部 Wiki の同期
