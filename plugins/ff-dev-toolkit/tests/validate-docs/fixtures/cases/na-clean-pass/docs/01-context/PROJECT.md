# PROJECT.md — Sprout

## ビジョン

水やりを忘れない。植物ごとの「次にいつ水やりすべきか」を一目で分かるようにする。

## 対象ユーザー

- 家庭菜園・観葉植物を育てる個人
- 水やり頻度を管理したいライトユーザー

## 主要機能

- 植物の登録
- 水やり記録
- 次回水やりのリマインド

## 非機能要件

- オフラインでも記録できる
- 起動から記録まで 3 タップ以内

## スコープ外

- 病害虫の診断
- 複数ユーザーでの共有
