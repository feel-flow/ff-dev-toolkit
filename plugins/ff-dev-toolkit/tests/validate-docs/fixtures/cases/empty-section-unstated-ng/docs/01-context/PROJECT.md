# PROJECT.md — Beacon

## ビジョン

障害の一次対応を遅らせない。アラートを見逃さず、正しい担当者へ最短で届ける。

## 対象ユーザー

- SRE / インフラ運用チーム
- オンコール当番のエンジニア

## 主要機能

- アラート集約
- 担当者ルーティング
- エスカレーション

## 非機能要件

- 通知遅延は 5 秒以内
- 可用性 99.9%

## スコープ外

- インシデントの事後分析レポート
- 監視系そのものの提供
