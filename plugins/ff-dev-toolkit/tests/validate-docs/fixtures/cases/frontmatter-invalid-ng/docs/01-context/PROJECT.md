# PROJECT.md — Nimbus

## ビジョン

雨に濡れない。朝の通知だけで傘の要否が判断できる状態を目指す。

## 対象ユーザー

- 屋外移動が多い社内メンバー
- 天気アプリを毎朝開くのが面倒なライトユーザー

## 主要機能

- 通知地点の登録
- 降水予報の朝次通知
- 閾値（降水確率）のカスタム設定

## 非機能要件

- 通知は毎朝 7:00 までに配信する
- 予報 API 障害時は前回値でフォールバックする

## スコープ外

- 週間予報の詳細表示
- 複数言語対応
