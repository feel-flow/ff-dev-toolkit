## プロジェクト概要

個人事業主向けの簡易会計記録アプリ。収支の記録とレポート出力を行う。

## 技術スタック要約

- FE: Svelte / BE: Go / DB: SQLite

## ディレクトリ構造

```
cmd/          # エントリポイント
internal/     # アプリケーションコード
docs/         # 仕様ドキュメント
```

## 守るべきルール

- **命名規則**: Go 標準（Exported は PascalCase、パッケージ内は camelCase）
- **コーディング規約**: エラーは握りつぶさず必ず返す

## 重要な制約

### 前提

- オフラインファーストで動作すること

### コード生成ルール

- 標準ライブラリを優先し、外部依存は最小化すること
