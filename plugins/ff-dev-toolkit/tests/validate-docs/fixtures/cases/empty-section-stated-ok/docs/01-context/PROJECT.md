# PROJECT.md — Ledger

## ビジョン

確定申告に必要な記録を、専門知識なしで日々ためられるようにする。

## 対象ユーザー

- 個人事業主・フリーランス
- 会計ソフトの操作に不慣れな層

## 主要機能

- 収支の記録
- カテゴリ別集計
- 年次レポート出力（CSV）

## 非機能要件

- 起動 1 秒以内
- データはローカル暗号化保存

## スコープ外

- 給与計算
- 複数ユーザーの同時編集
