# DOMAIN.md — Ledger

## ドメイン概要

会計記録ドメイン。中心となるエンティティは「取引（Transaction）」。

## エンティティ定義

- **Transaction**: 日付・金額・区分（収入/支出）・カテゴリ・メモを持つ記録単位
- **Category**: 取引を分類するラベル

## ビジネスルール

- 金額は 0 より大きい整数（円単位、小数なし）
- 収入・支出のいずれかを必ず持つ
- 削除は論理削除とし、レポートからは除外する

## 状態遷移

該当なし（取引レコードは作成後に状態を持たず、編集・論理削除のみ。ワークフロー状態はない）。

## ユビキタス言語

- **取引 (Transaction)**: 1 件の収入または支出の記録
- **区分**: 収入か支出かの種別
