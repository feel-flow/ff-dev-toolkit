# PROJECT.md — Orbit

## ビジョン

チームが「今やるべきこと」に集中できる、摩擦の少ないプロジェクト管理を提供する。

## 対象ユーザー

- 5〜20 名のプロダクト開発チーム
- 専任の PM を置かない小規模組織

## 主要機能

- タスク管理（作成・割当・状態遷移）
- マイルストーン管理
- 進捗ダッシュボード

## 非機能要件

- 主要操作のレスポンスは 300ms 以内
- 稼働率 99.5% 以上

## スコープ外

- 工数見積もりの自動化
- 外部 BI ツールとの連携
