# 社内在庫管理システム（架空・fixture 用）

## ビジョン

倉庫と営業が同じ在庫数を見て動ける状態を作り、欠品と過剰在庫を減らす。

## ターゲットユーザー

- 発注担当（倉庫部・2名）: 日次で発注判断を行う
- 営業担当（約15名）: 商談時に在庫を確認する

## 主要機能（現行）

- 在庫数の一覧表示・検索
- 入出庫の記録
- 月次棚卸しレポート

## 非機能要件

- 社内ネットワーク内でのみ利用
- 応答は一覧表示で3秒以内

## スコープ外

- 外部取引先への在庫公開
- 需要予測・自動発注
