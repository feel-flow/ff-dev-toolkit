# ドメインルール（架空・fixture 用）

## エンティティ

- **商品**: 商品コード（一意）、商品名、カテゴリ、標準発注ロット
- **在庫**: 商品コード、現在庫数、引当済数。**有効在庫 = 現在庫数 − 引当済数**

## ビジネスルール

- 在庫数は入出庫記録の登録時にのみ更新される（直接編集は禁止）
- 発注は発注担当のみが起票できる
- 月次棚卸しで実在庫と帳簿在庫の差異が出た場合、差異記録を残して帳簿を補正する

## 用語

- **欠品**: 有効在庫が 0 の状態
- **引当**: 受注済みだが未出荷の数量を在庫から確保すること

## 未定義事項（既知）

- 商品別の適正在庫水準・発注点は現状ルール化されていない（発注担当の経験による）
